Physics Models
==============

Physics models define material behavior and element-level computations for finite element analysis.

.. automodule:: pyFANTOM.Physics
   :noindex:

Available Physics Models
------------------------

* :class:`pyFANTOM.physics.LinearElasticity.LinearElasticity` - Linear elasticity physics model
* :class:`pyFANTOM.physics.NLElasticity.NLElasticity` - Nonlinear elasticity physics model
* :class:`pyFANTOM.physics.SteadyHeatTransfer.SteadyHeatTransfer` - Steady-state heat transfer physics model
* :class:`pyFANTOM.physics.LinearSymbolic.LinearSymbolic` - Symbolic linear physics model
